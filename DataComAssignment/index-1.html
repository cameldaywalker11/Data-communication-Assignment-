Course Overview

What is Data Communication?

Data Communication refers to the exchange of data between two or more devices via a transmission medium. It encompasses the hardware and software that enable us to send, receive, and process information across local and global networks. Below are the six foundational pillars studied in this course.

01
Fundamentals
Data representation, signals (analog vs digital), transmission modes, and the five components of a data communication system.
02
Transmission Media
Guided media (twisted pair, coaxial, fibre optic) and unguided media (radio waves, microwaves, infrared) — each with unique bandwidth and attenuation characteristics.
03
Network Models
The OSI 7-layer model and the TCP/IP model: how layers abstract complexity and enable interoperability between diverse hardware and software.
04
Error Detection
Parity bits, CRC, checksums, and Hamming codes ensure data integrity despite noise and interference on transmission channels.
05
Protocols
TCP/IP, HTTP, FTP, SMTP and more — standardized rules governing how devices initiate, maintain, and terminate communication sessions.
06
Network Security
Encryption (symmetric & asymmetric), firewalls, VPNs, SSL/TLS certificates, and threats like man-in-the-middle attacks and packet sniffing.

Historical Origins

The Telegraph — Birth of Data Communication

The electric telegraph, invented in the 1830s–1840s by Samuel Morse and Charles Wheatstone (independently), was the world's first practical long-distance data communication system.

It worked by sending electrical pulses along a wire: short pulses (dots) and long pulses (dashes) encoded letters and numbers using Morse code — the first digital communication protocol.

The key components included a battery as power source, a telegraph key (switch) operated by the sender, a long copper wire as the transmission medium, and an electromagnetic receiver/sounder at the destination that clicked out the coded message.

By 1866, the transatlantic telegraph cable connected Europe and North America — a feat that compressed message delivery from weeks (by ship) to mere minutes. This was the birth of global real-time data communication.

Detailed Study Notes

Key Topics Covered

Data Communication Basics

  • Definition and components of a data comm system
  • Simplex, Half-duplex, and Full-duplex modes
  • Analog vs Digital signals
  • Bandwidth, throughput, and latency
  • Noise and signal attenuation

Transmission Media

  • Twisted pair (UTP/STP) — categories 5e, 6, 7
  • Coaxial cable — baseband and broadband
  • Fibre optic — single-mode vs multimode
  • Wireless — 802.11 Wi-Fi standards
  • Satellite communication bands

OSI & TCP/IP Models

  • Physical, Data Link, Network layers
  • Transport, Session, Presentation, Application
  • TCP vs UDP — connection types
  • IP addressing — IPv4 & IPv6
  • Subnetting and CIDR notation

Data Encoding & Modulation

  • NRZ, Manchester encoding schemes
  • ASK, FSK, PSK, QAM modulation
  • PCM — Pulse Code Modulation
  • Nyquist and Shannon capacity theorems
  • Multiplexing: FDM, TDM, WDM

Error Detection & Correction

  • Single-bit and burst error types
  • Parity check (even/odd)
  • Cyclic Redundancy Check (CRC)
  • Hamming code for error correction
  • ARQ protocols: Stop-and-wait, Go-Back-N

Network Protocols & Security

  • HTTP/HTTPS, FTP, SMTP, DNS, DHCP
  • Symmetric encryption (AES, DES)
  • Asymmetric encryption (RSA, ECC)
  • TLS/SSL handshake process
  • Firewalls, IDS/IPS, VPN tunnelling
